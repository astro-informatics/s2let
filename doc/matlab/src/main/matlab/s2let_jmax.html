SYNOPSIS

DESCRIPTION 

 s2let_jmax
 Return the maximum scale for a wavelet transform

 Default usage:
        J_max = s2let_jmax(L, B)
    
    L is the band-limit for the transform,
    B is the wavelet parameter.
