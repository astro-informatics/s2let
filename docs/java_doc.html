Installing and using the Java framework

The bindings for Java were generated with JNAerator and BridJ. The corresponding jar includes all dependencies but is currently limited to MacOS. Adding this jar to the classpath is sufficient to use S2LET from java. Linux will be fully supported in a future release. Please contact Boris Leistedt if you need Linux support before this release.
